Logle is a web application that utilizes the Warcraft Logs API to quickly crawl through all dungeon logs of the requested player.
It then reports their highest keyLvl parses achieved per dungeon along with an average in the center block.
Only timed dungeons that are of equal or greater key level set by the minimum key level will be considered.

Specialization is not necessary, but will restrict the crawl to only that spec.
